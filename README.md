## 🧩 Key Features

### 🔹 **1. Database Creation**
- Extracts raw customer, product and sales information from CSV data  
- Cleans and standardises fields  
- Integrates datasets into a unified model  

### 🔹 **2. Star Schema for Analytics**
Includes:

- **Fact tables** (e.g., sales)  
- **Dimension tables** (e.g., customers, products,)  

### 🔹 **3. Exploratory Data Analysis**
Includes exploration of:

- Dates (first/last order date, customer lifespan) 
- Measures (total sales, total customers, average price)
- Categories (total revenue per customer, average cost per category, total customers per country) 
- Ranking (top 5 products by revenue, 3 customers with fewest orders)

### 🔹 **4 Advanced Data Analytics**
Covers:

- Change over time
- Cumulative
- Part to whole
- Segmentation
